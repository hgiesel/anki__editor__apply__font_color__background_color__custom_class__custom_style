This add-on allows you to quickly set a font color, the background color or a a custom class or custom style. You can use shortcuts, buttons from the top of the editor or the context menu. I frequently apply the same styling and want to quickly change between these. You can configure different colors, styles, or classes. For details about the configuration options see the readme that is shown in the add-on configuration dialog of 2.1 and the Styling Cards section from the manual. If you use Anki 2.1 and if you installed this add-on before 2019-02-18 and you have updated you need to reset the config of this add-on: In the lower left corner of the add-on configuration window click the button "Restore Defaults". If you use Anki 2.0 there is in the folder of the add-on a file named readme.md and a config.json that you need to adjust, for details see here. Some of the default shortcuts might not work on all systems (like MacOS) and might conflict with built-in or general shortcuts. Then you need to adjust the config. If you just want to sometimes change the text color or background color this add-on might be overkill. Or maybe you find the config too complicated. Or maybe you run into an error with this add-on. Then you could have a look at these options: - The option to change the background color is available in the Power Format Pack (2.0)/Mini Format Pack(2.1). But switching between different colors is time consuming. - Anki has a function to change the font color. But switching between different colors is time consuming. - There's the add-on Quick Colour Changing that allows to set shortcuts for different font colors. # styling with "class" This just applies a class. You need to set the appropriate CSS in the relevant stylesheet. This gives you much flexibility and makes it easy to adjust the styling for all cards you have - e.g. if you use some dark mode etc. You need to adjust TWO style sheets: - You need to adjust the stylesheet of the editor in the Add window and bottom of the browser with the add-on Customize Editor Stylesheet. - For reviews you need to adjust adjust the Card Styling. If you have many card types or note types it might be easier to use one external stylesheet that you use for each note type. You put this stylesheet in your media folder and in the styling section use "@import". The only content of the Styling section would be e.g. @import url("_style.anki.main.css"); Note the leading underscore in the file name. This is necessary to keep Anki from marking the file as unused and deleting it. for the location of the media folder of your profile see this section from the manual. # applying different font size If you want to change the font size you could apply a class and define appropriate css. I dislike setting fixed values like "30pt" in my cards because my next notebook might have a much higher resolution and who knows how such a fixed value will look on my next tablet.
